# **BudgetWise**  

### **Overview**  
BudgetWise is an open-source tool designed to help users better understand and manage their expenses. It prioritizes **privacy** by using local processing with small language models (SLMs) to analyze credit card statement PDFs, extract transactions, and display insights in a user-friendly format.

### **Key Features (Planned)**  
1. **Local Processing with SLMs**  
   - Analyze credit card statements locally to ensure privacy and avoid sharing sensitive data.  

2. **PDF Parsing and Data Extraction**  
   - Automatically parse PDF statements to extract transaction details, dates, amounts, and categories.  

3. **Expense Categorization**  
   - Auto-categorize expenses into groups (e.g., food, travel, subscriptions).  
   - Support for custom categories and user-defined tags.  

4. **Insights Dashboard**  
   - Visualize spending patterns with charts and graphs.  
   - Identify recurring expenses and subscription services.  

5. **Anomaly Detection**  
   - Highlight unusual or potentially fraudulent transactions.  

6. **Monthly Reports**  
   - Generate monthly summaries for easier budget tracking.  

7. **Cross-Platform Compatibility**  
   - A lightweight app accessible via desktop and mobile browsers.  

---

### **Tech Stack**  
- **Backend:** Python with Flask  
- **Frontend:** HTML, CSS, JavaScript  
- **PDF Parsing:** Libraries like `PyPDF2` or `pdfplumber`  
- **Local SLMs:** Integration with models like LLaMA or GPT-based small language models
